# 配置

模组配置可在模组菜单中的模组条目中找到

![modconfiglocation](/assets/images/ui/modconfiglocation.png)

该模组包是完全模块化的。虽然它默认已启用所有内容，但您可以在配置文件中关闭建筑或整个模组。

通过其类别中的“启用模组内容”复选框来启停整个子模组：

![config preview](/assets/images/ui/configpreview.png)

## 建筑编辑器

可以通过模组配置界面顶部的相应按钮来进入建筑编辑模式。

它允许您单独关闭该模组所添加的任何建筑物，并对其中一些建筑物进行设置配置。

具有可配置设置的建筑物会带有齿轮图标加以标识。

如果在主配置文件中禁用了某个子模组，那么该模组的所有建筑也将自动被禁用。那些之前存在于多个不同模组中的建筑，如果其中至少有一个模组处于启用状态，那么它们将保持启用状态。
