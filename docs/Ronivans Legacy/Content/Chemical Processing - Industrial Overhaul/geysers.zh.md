# 新 间歇泉
启用此模组配置将为游戏添加数种随机间歇泉，这将影响世界生成。若需使用指定种子地图，请关闭间歇泉配置选项以避免干扰。注意：关闭后这些间歇泉将不可用，除非通过其他模组或调试模板重新添加。
