    Rocket Port Adapters¶

    Rocket Port Adapters are a new type of building that attaches to the rocket platform.
    This new building type does not load anything on its own, but instead allows connecting rocket platforms that aren't placed directly next to each other.
    Multiple adapter buildings can be placed next to each other if there are larger distances to bridge.

    Insulated Rocket Port Wall Adapter¶

    Insulated Rocket
    Port Wall Adapter    		 An insulated wall adapter to seal off rocket start areas.
    Automatically links when built to the side of a Rocket Platform
    or another Rocket Port
    Sprite
    Building-ID RTB_WallConnectionAdapter
    Research Temperature Modulation
    Building Category Rocketry/RocketFueling
    Dimensions 1x2 tiles
    Decor -5 (Radius: 1 tile)
    Requires Must be built next to Rocket Platform
    Effects Fully insulated connection between rocket platforms
    Materials
    Raw Mineral 800kg
    Refined Metal 100kg

    Rocket Port Ladder Adapter¶

    Rocket Port
    Ladder Adapter    		 Connects adjacent rocket platforms while doubling as a ladder.
    Automatically links when built to the side of a Rocket Platform
    or another Rocket Port
    Sprite
    Building-ID RTB_LadderConnectionAdapter
    Research Smelting
    Building Category Rocketry/RocketFueling
    Dimensions 1x2 tiles
    Decor -5 (Radius: 1 tile)
    Requires Must be built next to Rocket Platform
    Effects Functions as ladder while connecting rocket platforms
    Materials
    Refined Metal 300kg
